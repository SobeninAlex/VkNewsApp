## VkNews

Приложение для просмотра ленты новостей в ВКонтакте

Для проектирования presentation слоя используется паттерн MVVM

## Stack

- MVVM
- Jetpack Compose
- VK Api
- Splash Api
- Retrofit
- OkHttp
- Dagger
- Navigation Compose
- Coroutines Flow
